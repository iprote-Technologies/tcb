# Prompt: Develop a Production-Ready Laravel Package for TCB Bank CMS API Integration

Develop a **production-ready Laravel Package** named **`iprote/tcb-cms`** that integrates with the **TCB Bank CMS API** for payment collection, instant payment notifications (IPN), reconciliation, reference number cancellation, loan disbursement, and other future CMS services.

The package must be designed as a reusable Composer package that can be installed into any Laravel application.

The implementation should follow Laravel package best practices and support Laravel **11+, 12+, and 13+**.

# Authentication

The package should **NOT** require account numbers inside the `.env` file.

Only API credentials should be configured in `.env`.

Example:

```env
TCB_API_KEY=
TCB_PARTNER_CODE=
TCB_BASE_URL=
TCB_WEBHOOK_SECRET=
TCB_TIMEOUT=30
TCB_VERIFY_SSL=true
TCB_LOGGING=true
TCB_QUEUE_CONNECTION=database
```

Everything else should come from the database.

Example tables

# Account Resolution

Whenever an API request is made, the package should automatically determine the correct account.

Example

Loan Application Fee

↓

Branch

↓

Default Collection Account

↓

Profile ID

↓

Send API Request

Loan Disbursement

↓

Branch

↓

Default Disbursement Account

↓

Transfer

The application using the package should never manually pass profile IDs unless overriding defaults.

# Important Business Rules

1. Only API credentials shall be configured in `.env`.

2. Branch bank accounts must be stored in the database.

3. One branch can own multiple TCB accounts.

4. The package must automatically resolve the correct account based on the transaction type.

5. The package must never hardcode account numbers or profile IDs.

6. All requests must be branch-aware.

7. All API operations must be logged.

8. Webhooks must be idempotent and safely reprocessable.

9. The architecture must support future TCB CMS APIs without breaking existing functionality.

10. The package should be generic enough to be reused by loan management systems, ERP systems, school management systems, utility billing systems, hospital systems, SACCO systems, and any Laravel application integrating with TCB Bank CMS services.
